# Change VueJS (4) to Reverse Proxy 

# 1. Clone

```sh
$ git clone https://github.com/jittagornp/docker-workshop.git  
$ cd change-vuejs-to-reverse-proxy   
$ cd frontend   
```

# 2. Deploy 
```sh
$ chmod 777 deploy.sh 
$ ./deploy.sh 
```

# 3. Test 

> http://<HOST_NAME or IP>  
> http://<HOST_NAME or IP>/api  
> http://<HOST_NAME or IP>/api/users
