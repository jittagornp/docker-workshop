# Build and Run Java Backend API (Connect to Mysql (2))

# 1. Clone

```sh
$ git clone https://github.com/jittagornp/docker-workshop.git  
$ cd build-and-run-java-api 
$ cd spring-boot-api 
```

# 2. Deploy 
```sh
$ chmod 777 deploy.sh 
$ ./deploy.sh 
```
